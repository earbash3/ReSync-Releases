## üéß Why ReSync

ReSync keeps Rekordbox aligned with your folder workflow in a compare-first, approval-based flow.

- Folder structure stays your source of truth
- Changes are visible before write
- Sync applies only what you approve

## ‚ö° Workflow

1. Open library
2. Compare folders vs. Rekordbox
3. Review preview
4. Apply sync

Fast, controlled, DJ-friendly.
